# TCS-NQT-Aptitude-System
This is a Python-based console application designed to help students practice aptitude questions for the TCS NQT exam.
## Features
- Multiple-choice aptitude questions
- User input validation
- Score calculation
- Console-based test system
## Technologies Used
- Python
## How to Run
1. Install Python
2. Download the repository
3. Run the file using:
   python aptitude_test.py
